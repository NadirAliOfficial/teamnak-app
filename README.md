# TeamNAK App

Official TeamNAK Flutter mobile app — Smart Tech. Real Impact. A hub for TeamNAK's tools, services, and client communication.

## Features
- Service showcase and portfolio
- Client project tracking
- Direct contact and messaging
- News and updates feed

## Tech Stack
- Flutter / Dart
- Firebase (auth, Firestore)
- Material Design 3

## Build
```bash
flutter pub get
flutter run
```
